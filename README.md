# Bon Voyage - マイクロドローン室内撮影サービス

横浜発のマイクロドローン室内撮影専門サービスのウェブサイトです。

## 🚁 サービス概要

マイクロドローンを使用した室内空撮サービスで、施設の魅力を最大限に引き出します。

### 対応施設
- サウナ施設
- ホテル・旅館
- 結婚式場
- 工場（リクルート動画）
- オフィス（来客促進動画）
- その他商業施設

## 📋 ウェブサイトの特徴

### デザイン
- モダンでプロフェッショナルなデザイン
- レスポンシブ対応（PC、タブレット、スマートフォン）
- ダークテーマを基調とした高級感のある配色
- スムーズなアニメーション効果

### セクション構成
1. **ヒーローセクション**: インパクトのある第一印象
2. **サービス概要**: 3つの主要な強み
3. **対応事例**: 6種類の施設事例
4. **ポートフォリオ**: Vimeo動画の埋め込み（3件）
5. **料金プラン**: 3つの価格帯（基本/スタンダード/プレミアム）
6. **資格・保険**: 国家資格と保険情報
7. **お問い合わせ**: フォームと連絡先情報

### 技術スタック
- HTML5
- CSS3（カスタムプロパティ、グリッドレイアウト、Flexbox）
- Vanilla JavaScript（ES6+）
- Google Fonts（Noto Sans JP、Montserrat）

## 🎨 カスタマイズ方法

### 1. Vimeo動画の設定

`index.html`の以下の部分を実際のVimeo動画IDに置き換えてください：

```html
<!-- サウナPR動画 -->
<iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID_1" ...></iframe>

<!-- 工場リクルート動画 -->
<iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID_2" ...></iframe>

<!-- オフィス来客促進動画 -->
<iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID_3" ...></iframe>
```

**Vimeo動画IDの取得方法**:
1. Vimeoにログインして動画をアップロード
2. 動画のURLから数字の部分を抽出（例: `https://vimeo.com/123456789` → `123456789`）
3. `YOUR_VIDEO_ID_1`などを実際の数字に置き換え

### 2. 連絡先情報の変更

`index.html`のお問い合わせセクション（`#contact`）で以下を更新してください：

```html
<p class="contact-phone">📞 あなたの電話番号</p>
<p class="contact-email">✉️ あなたのメールアドレス</p>
```

### 3. カラースキームの変更

`style.css`の`:root`セクションでカラー変数を調整できます：

```css
:root {
    --primary-color: #1a73e8;      /* メインカラー */
    --secondary-color: #ff6b35;     /* アクセントカラー */
    --accent-color: #00d4ff;        /* 強調カラー */
    /* ... */
}
```

### 4. 料金プランの調整

`index.html`の料金セクション（`#pricing`）で価格や内容を変更できます。

## 🚀 デプロイ方法

### GitHub Pagesへのデプロイ

1. GitHubリポジトリを作成
2. ファイルをプッシュ
3. Settings → Pages → Source を `main` ブランチに設定
4. 公開URLにアクセス

### その他のホスティングサービス

以下のサービスでも簡単にホスティング可能です：

- **Netlify**: ドラッグ&ドロップでデプロイ
- **Vercel**: GitHubと連携して自動デプロイ
- **Cloudflare Pages**: 高速なCDN配信

## 📝 フォーム送信の実装

現在、フォーム送信はデモ実装です。実際に機能させるには以下のオプションがあります：

### オプション1: フォームサービスの利用

**Formspree**（推奨）:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
    <!-- フォームフィールド -->
</form>
```

**Netlify Forms**（Netlifyでホスティングする場合）:
```html
<form name="contact" method="POST" data-netlify="true">
    <!-- フォームフィールド -->
</form>
```

### オプション2: バックエンドAPIの構築

Node.js、PHP、Pythonなどでバックエンドを構築し、`script.js`のフォーム送信処理を実装します。
