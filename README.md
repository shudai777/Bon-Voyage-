横浜発のマイクロドローン室内撮影専門サービスのウェブサイトです。

- カフェ・飲食店
- その他各種施設

### フォーム設定方法（5分で完了）

#### 【推奨】Formspree（無料・簡単）

**1. Formspreeに登録**
- https://formspree.io/ にアクセス
- 「Get Started」をクリック
- メールアドレスで無料登録

**2. 新しいフォームを作成**
- ダッシュボードで「+ New Form」をクリック
- フォーム名: `Bon Voyage お問い合わせ`
- 受信メールアドレス: `www.we9999@gmail.com`

**3. フォームIDを取得**
- 作成されたフォームの**ID**をコピー
- 例: `xvoepqra` のような英数字

**4. index.htmlを修正**
```html
<!-- 現在（452行目付近） -->
<form class="contact-form" id="contactForm" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">

<!-- 修正後 -->
<form class="contact-form" id="contactForm" action="https://formspree.io/f/xvoepqra" method="POST">
```

`YOUR_FORM_ID` を実際のIDに置き換えてください。

**5. 完了！**
- GitHubにプッシュすれば、すぐに問い合わせを受信できます
- 問い合わせがあると `www.we9999@gmail.com` にメールが届きます

### その他のフォームサービス

#### Netlify Forms（Netlifyで公開する場合）
- 設定が非常に簡単
